Request Form Usage

Step 1 (Install needed dev tools)
 	- Python
 	- SQLAlchemy
 	- Flask

Step 2 (Start server)
 	1: In command prompt, locate the directory the file was downloaded to. 
	2: Run "python app.py" to serve the application.

Step 3 (Enter App)
	1: Open a web browser
	2: In the URL bar, enter "localhost:5000/main"

Step 4 (submit form)
	Fill out the form and submit.
